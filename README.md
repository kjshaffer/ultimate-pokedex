# Ultimate Pokedex

CS4610 Modern Web Development – Assignment 1

This project is a Next.js + TypeScript web application that consumes the PokeAPI to create a full Pokédex experience.

## Live Deployment

https://ultimate-pokedex-three.vercel.app/pokemon

## Features

- Searchable lists for Pokémon, Locations, Moves, and Generations
- Individual detail pages for each entity
- Displays Pokémon stats, sprites, encounter locations, and moves
- Displays location regions, areas, and Pokémon encounters
- Displays move stats, flavor text, and Pokémon that can learn the move
- Displays Pokémon by generation

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- PokeAPI (https://pokeapi.co/)
- Vercel (Deployment)

## Running Locally

Install dependencies:

npm install

Run development server:

npm run dev

Build production version:

npm run build

Start production server:

npm run start
